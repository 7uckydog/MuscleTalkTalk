## 프로젝트 설계주안점
+ 운동 계획과 운동 결과를 캘린더로 제공
+ 다른 사용자가 올린 루틴을 가져오거나 또는 내 루틴을 공유
+ 구체적인 운동을 원하는 사용자를 위해 트레이너 연결 시스템을 제공
+ 운동을 보다 쉽고 간편하게 할 수 있도록 도와주는 웹 사이트
